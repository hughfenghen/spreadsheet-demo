Whether to generate columns automatically while binding to a data source.
In This Topic
    autoGenerateColumns Method
    In This Topic
    Gets or sets whether to generate columns automatically while binding to a data source.
    Syntax
    • Javascript (Usage)
    • Javascript (Specification)
    var instance = new GC.Spread.Sheets.Tables.Table(name, row, col, rowCount, colCount, style, options);
var returnValue; // Type: any
returnValue = instance.autoGenerateColumns(value);
    function autoGenerateColumns( 
   value : boolean
) : any;

    Parameters

    value
    Whether to generate columns automatically while binding to a data source.

    Return Value

    If no value is set, returns whether to generate columns automatically while binding to a data source; otherwise, returns the table.
    Example
    This example binds a table and sets the autoGenerateColumns method.

    this.name = name;
    this.logo = logo;
    this.slogan = slogan;

    this.city = city;

function Customer(id, name, company) {
    this.id = id;
    this.name = name;
    this.company = company;
}

function Record(description, quantity, amount) {
    this.description = description;
    this.quantity = quantity;
    this.amount = amount;
}

function Invoice(company, number, date, customer, receiverCustomer, records) {
    this.company = company;
    this.number = number;
    this.date = date;
    this.customer = customer;
    this.receiverCustomer = receiverCustomer;
    this.records = records;
}
